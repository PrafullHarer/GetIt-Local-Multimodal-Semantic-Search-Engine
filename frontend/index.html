GetIt — Local Multimodal Semantic Search Engine

A privacy-first, offline-capable semantic image search engine. Search your local image library using natural language, powered by hybrid AI combining visual understanding (CLIP) and semantic captions (BLIP).

Features

  • Hybrid Search: Combines visual embeddings (CLIP) with generated captions (BLIP).
  • Smart Keyword Cleaning: Automated noise reduction for accurate matching.
  • 100% Offline: No cloud, no API keys — your data stays local.
  • Auto Indexing: Captions & embeddings generated automatically.

Tech Stack

  • Backend: Python, FastAPI, PyTorch
  • AI Models: OpenAI CLIP, Salesforce BLIP
  • Frontend: Vanilla HTML / JS / CSS
  • Storage: JSON-based index (portable & simple)
